Here are some things about Git:

  • git init - this makes a new Git repository
  • git status - this tells you some information about the repository
  • git add - this adds a file for Git to track
  • git commit - this takes a snapshot of the project
  • git push - this uploads the repository to a remote
  • remote - this is another location e.g. on the internet that your Git repository is stored
